# 📋 To-Do List

A simple and convenient task management app built with vanilla JavaScript. It allows you to add, delete, and mark tasks as completed. All data is saved in the browser's `localStorage`, so your to-do list persists after page refresh.

## 🛠️ Technologies Used

*   HTML5
*   CSS3
*   JavaScript (ES6+)
*   Local Storage API

## ✨ Features

*   Add new tasks.
*   Mark tasks as completed / uncompleted.
*   Delete individual tasks.
*   Automatic saving of all tasks in the browser.

# 📋 Список задач (To-Do List)

Простое и удобное приложение для управления задачами на чистом JavaScript. Позволяет добавлять, удалять и отмечать задачи как выполненные. Все данные сохраняются в `localStorage` браузера, поэтому список задач не пропадает после перезагрузки страницы.
